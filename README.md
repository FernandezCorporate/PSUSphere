# PSUSphere

## Project Description
PSUSphere is a web application designed to help organizations within Palawan State University in managing their student members. 

## Features
- User-friendly Admin page.
- Enhanced accessibility via search and filter functionalities.
- Supports CRUD functionalities for managing records.
